Welcome to Wikonomi

Want me to show you the ropes?

Wikonomi is a community-built database for useful local information in Papua New Guinea: products, prices, businesses, practical guides, and transport options.

Search before you spend Share what you find Help keep PNG information current

1. Find local information

Search products, prices, businesses, and guides

Start from the homepage search box. Search for an item, brand, shop, location, or how-to topic, then compare the latest reports, dates, photos, ratings, and map locations.

Try the search

2. Add a price

Turn something you saw in a shop into useful data

Tap Post → Post Prices, then add the product, price, business, branch or location, and a recent photo where possible. Accurate package sizes and dates matter.

Add your first price

3. Add a business

Help PNG businesses become easier to find

If a shop, service, organisation, or branch is missing, use Post → Post Business. Add the correct name, category, contact details, image, and location before linking prices or guides.

Add a business

4. Create a practical guide

Document how things actually get done in PNG

Guides turn local experience into reusable steps—such as getting a licence, completing a process, or using a service. Add clear steps, useful links, photos, and the relevant business or organisation.

Create a guide

5. Find local transport

Browse verified drivers and continue on WhatsApp

The Transport Index lets you filter verified cabs by vehicle type, home area, availability, and distance. Choose a driver, review the profile, then contact them through WhatsApp.

Browse transport
